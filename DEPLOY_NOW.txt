FIREBASE DEPLOY COMMANDS
========================

Since you have firebase.json open, you are in the correct directory!

Copy and paste these commands one by one into Command Prompt:

1. npm install -g firebase-tools

2. firebase login

3. firebase use gernboke

4. firebase deploy --only hosting

After successful deployment, visit:
https://gernboke.web.app

OR simply double-click:
simple-firebase-deploy.bat

Firebase Console:
https://console.firebase.google.com/project/gernboke
